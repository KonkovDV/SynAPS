# Maintainers

## Current Maintainer Team

- Primary maintainer: KonkovDV
- Repository: https://github.com/KonkovDV/SynAPS

## Responsibilities

Maintainers are expected to:

1. preserve deterministic behavior and contract compatibility;
2. enforce reproducibility and evidence-backed public claims;
3. review security-sensitive changes in workflows, dependencies, and external I/O paths;
4. keep governance files and release policy aligned with actual repository behavior.

## Escalation

For security reports, use the private disclosure path defined in SECURITY.md.
For high-risk release decisions, require at least one maintainer approval before tagging.
